【甲子民风】惠来400余年历史的山美古寨

Posted on 七月 2, 2017 本文总阅读量次
三甲人最喜欢访问的网站

从惠来县隆江镇东上县城惠城镇，走葵和公路，5里处是古墟迹苗圃，转西北行3里有村名山美村，山美村东双溪怀抱——罗溪北南流至此掉头南北流，山美溪北南流，双溪汇聚于东墩山东畔双溪嘴下，转东流。东墩山形如倒放的大鼎，上建古寨，古称东墩寨，现称山美古寨，远近闻名。 山美古寨占地约500亩，明万历年间兴建，天启二年（公元1622年）完工，有400余年历史了，是惠来县面积最大的古寨。 嘉靖年间，山美古寨创建者严侃素从现在的福建漳州南靖县山城镇山边村严厝社南迁至此，原居住在罗溪东边井美村吴爷公山北面鸡公树脚，在邻村凤光村做私塾先生。侃素公系孤单外乡人，备受井美村人欺凌。凤光村东家感于先生教子之功，献2块地供先生选择居住，一是留山，二是东墩山。
侃素公请风水先生看地，断言：留山富贵少人丁，东墩山人丁兴旺少富贵。侃素公看中东墩山南北双龙入首，东畔双溪汇合，风水清奇，风光绮丽，说：人丁兴旺必有人才出，富贵也可期。遂选择东墩建寨。山美村现有人口近万人，都系严姓，八成系侃素公后裔，繁衍已至十七世，系惠来县葵和公路北面最大的村落，人才辈出，乡贤众多，印证了侃素公的高瞻远瞩。

经过两代人的努力，古寨围筑完成，呈3街6巷布局。寨内中间一条长巷横贯南北，东西两向每向6条巷，每巷6座下山虎。东西直巷拐曲，在中间横巷处错位交接，叫做“十字街”。寨墙内一方圆形的长巷首尾相接，绕寨一周，与东西直巷都联通。
长巷外边建正方形房子做寨墙，房门朝寨内，朝寨外有小窗户，便于侦探敌情和防御。原寨墙开东西南北四个寨门，后来北门因故封堵。说法有二：一说开四个寨门的寨子应有寨里人做县令以上官职，那时山美村还没有出现这样的人物；二说东墩龙脉自北往南又回头北走，出北门，聚北门外，先人怕龙气外泄，就封堵北门，还在北门外开挖一大片露天厕所，以堵龙气外泄。笔者以为第二种说法更符合先人的想法。 东门名“吞长江”，东起鳌峰，南朝雁塔，西绕青山，秀气来临，左右两溪水流来汇合；西门名“衔远山”，马褂鞍，狮望月，水为带，沟如襟，四周峰峦，万顷明塘来朝拱；南门名“湧巨川”，座上近神，宫钟敲响，面前园林布景，北朝双过年峰，似寻母腹，水朝纳。

山美古寨建筑布局处处透出丰富的风水内涵，可见古人对风水的讲究。
建寨以来，古寨一直是山美村民主要的居住地。随着寨里人口的增多，一些寨里人迁出寨外，在东墩山四周建房居住，但古寨一直人气鼎盛。近十几年来，迁出寨外建房居住的人越来越多，现在寨里居住的大多是老年人。 400余年的风雨洗礼，山美古寨保存完好，居住、生产、生活功能长盛不衰。石灰与沙子混凝筑就的寨墙和屋墙坚固，少有倒塌，青苔色浑厚；石板路光滑如镜，几可照人，一切都呈现出一种独有的古韵。走在寨里，如果不是房前屋后的电线、网线、自来水管或古屋里传出来的电视声、电话声，简直就是回到了400年前的明朝村庄。 山美古寨，是创建者的丰碑，是子孙后代的根，是先人留给后人的宝贵遗产。
