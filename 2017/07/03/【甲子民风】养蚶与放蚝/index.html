【甲子民风】养蚶与放蚝

Posted on 七月 3, 2017 本文总阅读量次
三甲人最喜欢访问的网站

蚶，是珠蚶的简称，为人们喜爱的海产品，又名“血蚶”。饶平县碧洲岛拥有广阔的浅滩涂，特别适合珠蚶的生长，因此养殖珠蚶也成为村民传统的作业，幸运者丰产丰收，常有几倍厚利可得。清代岛上贡生郑玉壶曾作一首竹枝词：年年生活放虹栽，养得蚶来利几倍。郎若卖蚶休卖尽，留成老蚌结珠胎。词中描述养珠蚶的利润和方法，令读者很是羡慕。 珠蚶养殖大多是早春就开始放养蚶苗，蚶苗小如针头，跟海沙混在一起很难辨别，俗称“蚶沙”，通常从外地购买。蚶苗放下去，要时常关照，提高其成活率。一是防止蟹、蛴、螺等吞食，二是要经常帮它移动位置，让蚶能吃到更多微生物。蚶民经常使用一种自制工具：一根长竹竿，竿尾横穿一根约有一米长的木棍，木棍上钉了很多铁钉，像狼牙棒一样。每当潮退滩现，蚶民双手握住竹竿让木棍在滩面上来回扫动，这样既能帮助蚶翻动移位，同时还能捕到蟹、蛴、螺。珠蚶在主人的关照下，吃得多长得快，不出一年就从小得很难辨识生长为成蚶可以上市了。但捞上来的蚶不可洗得太干净，要略带些泥浆，才可停放一段时间。

一年生的蚶便可收获，三年生的老蚶不仅个头大还能结珠胎，价格比成蚶高出好几倍，但有些蚶民还舍不得卖，因为它可以继续繁殖珠蚶，服务生产。有些放养多年的老蚶捞起来后，掰开壳，有珍珠含在肉里，一粒粒亮晶晶，熠熠发光，更是人们喜爱的奇货。冬至后至清明前这个季节的蚶长得最肥，人们用近沸的水烫泡，然后掰开，就能见到鲜红如血的肉汁，吃起来肉脆爽口。
其营养丰富，味道非常鲜美，百食不厌，最是补益身体，是补血佳品。珠蚶肉不但好吃，其意头也好，潮汕民间俗称“合利钱”。人们喜欢将其作为祭品，或与猪肉、鸡蛋合在一起作为三牲贡品祭祖拜神，祈求祖宗、神明庇佑财源广进。年终岁首节日祭拜活动多，珠蚶自然成为抢手货，销量很大，有时供不应求。 一方水土养一方人。碧洲岛的另一主要滩涂海水养殖业是放蚝。这里的蚝个体小，不同于我们在市面上常见的吊蚝、大蚝，它生长在小石块上，由野生的成熟蚝把肉眼看不见的种苗吐出来，随海水漂流，遇到石面粘住变成蚝苗，然后慢慢成长，俗称“蚝石蚝”、“珠蚝”。 村民从陆上搬运条形小石块到滩涂摆放，每三四块靠在一起，形成中间留空的一锥形石堆，就成了蚝石。蚝石在流动的海潮中粘上蚝苗，等到蚝苗长大，蚝民们用铁凿把它敲下来，这就是人们说的“打蚝”。而石头还会再次粘上种苗继续生长，这样周而复始，一次投入，连年收获，可谓一本连收。但这些蚝石要经常修整摆放，不然就会被潮水冲倒散落，生长的蚝会被闷死，或影响蚝的成长。
而蚝石的摆放是有学问的，对于倒下的石块，蚝民们用一把二爪的铁钩，右手握柄使铁钩勾住石块的下方，左手抓住石的上方，合力把它提起来，重新摆放好。有经验的人会根据海水的流动方向，把石堆的间隔留得适当，既能让海水自由流动，又不致轻易把石堆弄倒。而对于滩涂土质较软的地方，石头放下很快就没入泥中，这就必须先用铲把海泥堆筑成一畦畦，然后等待每天潮退滩现，太阳晒海风吹，畦表面的泥涂日渐凝结变硬，一段时间后，畦面终于能承受石块的重量，才开始在畦上摆放小条石。

不过畦在海水的浸泡和流动下，会慢慢烂下来，逐渐由高变低。放在畦面的石头就会陷下去很难挖出来。这就需要在退潮时，经常用铲把畦沟的泥填上畦顶再让它风吹日晒变硬，然后移动蚝石，保持畦高涂硬。蚝畦在人们的呵护下，排列有序，大小相当，一眼望去很是壮观。 蚝石放置一年后，便可见蚝了，第三年便可获利。珠蚝收成期较珠蚶长，从农历十月至翌年四月都可采收，以春节期间最好，这时蚝长得肥，用它煮汤、泡粥、做蚝烙，都非常鲜甜可口。

蚝肉味道鲜美，打蚝撬蚝却不容易。冬春大潮多在早上，特别是农历正月初一初二早水大潮，潮水退得早，打蚝的人早早起床烧火煮饭，吃饱穿暖，挑起工具，冒着寒风赶到蚝畦，趁天未亮，就开始摸着石头，把蚝敲下来。虽然戴着手套，但有时锋利的蚝壳还是会划破手指，似刀的寒风会吹痛脸面，冰冷的海水会冻裂手脚。敲下来的蚝，称之为“蚝头”，要用一把铁锥扎进蚝嘴，掀开壳，将肉刮下来，这才是可供食用的蚝肉。
这种劳动也不容易操作，找不到蚝嘴难以撬开，撬开了刮得不好会把整块肉搞碎，弄得一塌糊涂。有时一不小心就会被锥扎伤手，或被壳割破皮，十指连心，痛得人额冒冷汗。其中艰辛只有生产者自己知道。 随着围海造田，碧洲岛变成了一个与大陆相连的海滨渔村，昔日的浅滩涂部分已变成稻田、渔塘，海水养殖的规模和形式也发生了翻天覆地的变化。珠蚝与珠蚶的养殖没有了以前的规模，而由于沿海的污染，产量也明显下降，即使价格不菲，但也渐渐淡出人们的视野了。如今，每当在市场上看到它们，就会想起家乡。
