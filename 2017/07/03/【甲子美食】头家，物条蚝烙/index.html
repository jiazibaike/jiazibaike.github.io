【甲子美食】头家，物条蚝烙

Posted on 七月 3, 2017 本文总阅读量次
三甲人最喜欢访问的网站

蚝烙（潮汕话念作o luah)是广东省潮汕汉族特色小食， 但它还不是潮汕地区独有的美食，在闽南地区和台湾地区，有着和蚝烙十分相似的美食“蚵仔煎”，“蚵”在潮汕话、闽南话中指的正是“蚝”，而它的发音正式“哦”，假如碰到一个潮汕人总是回复你“哦“，千万不要单纯的以为是在敷衍你，说不定他是想吃了。。。

（漫画设计：喃东尼）
在潮汕城市乡村小市集，常有这种小店或摆摊小贩。这种蚝烙，是用地瓜粉溶于水，拌葱珠，在一个平底的铁锅上煎，加上海蛎，再下蛋花，取起蘸鱼露吃。
鲜美酥香，酥而不硬，脆而不软，这是潮汕特色美食蚝烙的味道。新鲜煎好的金黄色蚝烙香气扑鼻，醮着一点点鱼露放入口中，鲜甜的味道随即在舌尖上蔓延开来，让人忍不住直呼“美味”

蚝烙是潮汕久负盛名的传统民间小吃。蚝烙实际上可以说是蚝煎，因为潮汕地区的“烙”，实际上是潮州菜烹调方法中的“煎”。
“蚝烙”这款传统潮汕小吃，有着很长的一段历史。相传早在清代末年，潮州城镇各地，制作“蚝烙”的小食摊已经十分普遍。其中最有名的，应为民国初年，位于潮州府城开元寺古井西北的泰裕盛老店。
这一小食铺，专门经营“蚝烙”，具制作的“蚝烙”特别好吃。原因是因为泰裕盛老店，在选料上十分严格，专门选取饶平汫洲出产的珠蚝，采用优质雪粉，甚至连猪油都要用本地猪的鬃头肉煎出来的猪油，制作的每一步骤十分考究，其煎制的“蚝烙”，具有特别鲜美的蚝香味，口感酥而不硬，脆而不软。由于泰裕盛老店制作的“蚝烙”口味特别诱人，故在当时名噪整个潮州地区，持续达半个世纪之久。

那么美味的蚝烙要怎么做呢，大家不妨动手试试看：
首先需要准备到以下这些材料：
红薯粉100g、珍珠蚝250g、鸡蛋2个、油、鱼露、香菜、葱少许

蚝内加入盐，不加水揉搓（注意力度不能把蚝弄破）

用流动的水冲洗干净沥干

薯粉加入适量的水，搅拌均匀

将葱切粒和生蚝加入到搅拌均匀的薯粉水中

平底锅烧热，加入适量的油

旺火，倒入香葱生蚝薯粉水

煎至略微成形，改中火

将搅拌好的鸡蛋液淋入一半

鸡蛋液凝固后翻面

翻面后再淋上余下鸡蛋液

煎至两面金黄即可出锅，盘底可以垫上生菜，装盘后用香菜点缀

开吃啦

小贴士
1.红薯粉和水的比例是1:3
2.整个过程不需要下盐，最后蘸鱼露即可，享用原汁原味的鲜美
3.开始下锅的时候，要旺火，才能迅速锁住生蚝的水分
4.香菜既是点缀，又是合味，蚝烙卷点生菜蘸鱼露，特别美味
5.“烙”即为煎的意思，不同于下很多的油去炸，烙出来的口感必须是外脆里嫩的，而非炸出来的脆脆的口感。
6.要趁热吃。
