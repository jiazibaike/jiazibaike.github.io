甲子览表交界.抓到这群人...

Posted on 三月 28, 2017 本文总阅读量次

3月25日，在省公安厅统一指挥协调下，汕尾市公安局相关职能部门合成作战，分工协作，精心经营，与揭阳惠来警方同步收网，先后在惠来、陆丰等地抓获涉毒犯罪嫌疑人11名，缴获制毒原料氯代麻黄碱450公斤，扣押涉案车辆3辆，摩托车3辆。

近日,汕尾公安禁毒合成作战专班民警在对掌握的涉毒线索进行梳理排查时，发现了“0221制贩毒品案”的相关案件线索。汕尾市公安局主要领导对此高度重视，组织专班立即召开案情分析会，并将相关信息上报省公安厅，同时通报揭阳警方。在省公安厅的指导下，专班与揭阳警方紧密联动，通过前线民警连日来的侦查作战，终于摸清了涉案人员的落脚点，掌握了其活动规律。

3月25日，汕尾公安禁毒合成作战专班民警掌握到该贩毒团伙将于当日下午进行交易的信息后，遂协调汕尾特警、边防、陆丰公安禁毒等部门组成专案警力，与揭阳警方展开联合收网。当日下午，两地警方在揭阳惠来抓获9名犯罪嫌疑人，缴获制毒原料氯代麻黄碱450公斤。此时，另有2名犯罪嫌疑人往陆丰方向逃跑。得知消息后，专班领导随即调配人员在陆丰进行设卡拦截，并将掌握的涉案人员信息及时反馈给前线队伍，民警最终成功在惠来与陆丰甲子交界的览表桥附近，将意图逃跑的2名犯罪嫌疑人汪某鑫（男，22岁，陆丰甲子人）、吴某彬（男，22岁，揭阳惠来人）抓获，并将2人移交揭阳警方。

目前，该案件由揭阳警方进一步侦办中，汕尾警方将积极配合揭阳警方下一步的调查取证工作。
