今天（2016年9月29日），陆丰市甲子镇第十六届人民代表大会第一次会议第二次全体会议在庄严的国歌声中胜利闭幕。

大会期间，出席会议的代表听取和审议通过了《政府工作报告》《人大主席工作报告》，审议通过《关于调整甲子镇居民住户环境卫生服务费标准的方案（草案）》。
今天上午，大会以无记名投票选举产生了新一届镇政府领导班子。唐本利当选为甲子镇人大常委会主席，赖豪当选为甲子镇镇长，卢金链、李祖创、余伟彬、余志昂为甲子镇副镇长。
会后，代表们份纷表示，2016年是”十三五”规划开局年，新一届镇班子同心协力，真抓实干，带领全镇人民开拓进取，有所作为，为建设一个平安、和谐、幸福、美丽的甲子镇作出贡献。
