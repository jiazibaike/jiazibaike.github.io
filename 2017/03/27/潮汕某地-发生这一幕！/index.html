潮汕某地.发生这一幕！

Posted on 三月 27, 2017

网友爆料：3月23日下午，海丰县交警在二环南路万和加油站路段，对无牌无证等交通行为进行了整治。在整治过程中，一名三轮车司机拒不配合，推搡、殴打执法人员，并持刀袭警。事发后，现场多名交警上前将其制服。


事后，海丰县公安局通过官方微博进行通报。


详细通报如下：
3月23日17时许，海丰公安交警大队民警在二环南路万和加油站路段查扣一部无牌证载货三轮车时，遭三轮车驾驶人暴力抗法。期间，此人情绪激动，先后用头盔、板手、螺丝刀等砸打、追赶执勤民警，又驾三轮车冲撞民警，随后驾三轮车逃离现场。目前，我公安机关对此人正全力追缉中。
在此提醒广大市民朋友，配合公安机关和相关部门的交通整治工作，如若出现暴力抗法、妨碍公务、扰乱社会秩序的行为，公安将坚决依法予以处理。
