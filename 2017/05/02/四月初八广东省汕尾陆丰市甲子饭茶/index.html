四月初八广东省汕尾陆丰市甲子饭茶

Posted on 五月 2, 2017 本文总阅读量次
三甲人最喜欢访问的网站

四月初八这一天以“饭茶”为最。

那一天不仅是迎宾接待时必备的佳品，而且是一种本土的民俗。饭茶并非是一般清茶，而是一种多味俱全的饭茶。

所谓‘饭茶’，又何为‘饭茶’顾名思义就是饭和茶，起初多意味着粗茶淡饭，后来演变得多色多味，平素里又显得奢华又有内涵。

系隔一、两个小时的白米饭，因刚开锅时其米饭较软，歇凉一会儿，饭就变得较有弹性成颗粒状。一般主选用乌龙茶科；铁观音。也可人曾用番石榴叶、四枝苦刺心等，前者略热后者凉，铁观音茶半发酵同时又俱有香、甘等味。

随着个人喜好则可另加辅料，一般辅料多以盐、九层塔、香菜、芹菜、小葱、香菇、绿豆（赤豆、白米豆）、花生米。将各式各样的青菜清洗干净并切碎，热锅下油，把青菜炒熟放少许盐即可，香菇是用热水泡一会后将其切开，也是热锅下油，又将豆类洗后用水煮熟放起。炒花生是一门较为讲究的活，要混合盐粉均匀翻炒至熟，筛去盐粉，再把一颗颗的花生搓成两半去外膜，这时单一味也是脆香无比。食用时先将乌龙茶叶用水煮后泡入饭中，伴入香菜、九层塔等，再配上炒熟的花生米，清脆爽口香。虽说工繁技琐但也是乐在其中。

据闻，四月初八是释迦牟尼佛诞即佛教当中的浴佛节。明朝时民间有食“结缘豆”的风俗，此传至清代极为流行，古书记载：“京都浴佛日，内城庙宇及满洲宅第。多煮杂色豆，微渡盐鼓，以豆萝列于户外，往来摄食之，”名为《结缘豆》。

这可能添加豆类的主要关系吧，因旧时人们的生活相对较为贫苦，祈望得到合家安康、生活富裕的一种美好的祝愿。

饭茶由菜谷豆类组成的既能补脾益胃，又能充饥解渴，清暑解毒。

吃上一碗，那浓浓的乡情和乡味在你身边萦绕着，同时也是象征吉祥平安永久常伴。
